# Discoveries

This folder is the heart of The Trail Project.

Scripts evolve.
Scenes change.
Dialogue gets rewritten.

But discoveries should be preserved.

Every discovery recorded here comes from honest conversations, real memories, and lived experience.

We don't collect ideas.

We collect discoveries.

Because every honest film begins with one.

## Working Rule

Conversation → Discovery → GitHub → Screenplay

The screenplay is the result.

The discoveries are the roots.
